#hubot-script

A hubot script for something

## Installation

Add **hubot-...** to your `package.json` file:

```json
"dependencies": {
  "hubot": ">= 2.5.1",
  "hubot-scripts": ">= 2.4.2",
  "hubot-...": ">= 0.0.0"
}
```

Add **hubot-...** to your `external-scripts.json`:

```json
["hubot-..."]
```

Run `npm install hubot-...`
